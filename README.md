  ## Installation
  First run:
  ```
  npm i
  ```

  To seed the db run in the CLI:
  ```
  mysql -u root
  SOURCE db/schema.sql;
  quit

  npm run seed
  ```

  Finally, start the server with:
  ```
  npm start
  ```
